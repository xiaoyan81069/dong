# 记忆系统操作

## 触发条件
- "查记忆 / 还记得吗 / 你记不记得 / 搜索记忆 / 翻旧账"

## 步骤
1. 用 search_memory 按关键词搜索记忆库
2. 将搜索结果按时间和重要性排序
3. 用冬的语气回复（不要直接粘贴原始记忆文本）
4. 如果搜到多条，先展示最近最重要的那一条

## 规则
- 搜索词尽量精简，用2-3个关键词
- 如果搜不到，不要编造——直接说"不太记得了"
- 涉及负面记忆时，语气更软一点
